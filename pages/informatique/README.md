# Documentation Page

## Overview
The Documentation page provides quick access to technical documentation for key technologies used in our projects.

## Technologies Covered
- Next.js
- Vercel
- Supabase
- Node.js
- React
- Tailwind CSS
- OVH
- GitHub

## Features
- Responsive grid layout
- Aesthetic technology cards
- Direct links to official documentation
- Hover effects for improved interactivity

## Card Design
Each card includes:
- Technology logo
- Brief description
- External link icon
- Unique color scheme

## Responsive Design
- Mobile: 1 column
- Tablet: 2 columns
- Desktop: 3-4 columns

## Logo Management
Logos are dynamically loaded from the `/public/logos/` directory. Use the `generate_logos.sh` script to update logos.
